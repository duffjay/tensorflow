Ubuntu 16.04

cuDNN 7.0.5
- install runtime deb
- then instell developer

### setup
```
mkdir ~/tf-demo
cd ~/tf-demo
python -m venv tensorflow-dev
```

### resume
```
cd ~/tf-demo
source tensorflow-dev/bin/activate
python3
```
